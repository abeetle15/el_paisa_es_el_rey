Y yo soy mas paisa que una tasada de masamorra con dulce macho, el paisa no tiene amigos el paisa tiene parceros, no conquista una mujer sino que le echa los perros, el paisa no tiene amante el paisa consigue mosa, y el paisa no se emborracha, se rasca hasta las pelotas, el paisa no se enamora el paisa solo se encoña, no se echa una cana al aire se le vuela a a la señora, el paisa no se enamora el se traga hasta la nuca, y una mujer no esta buena sino que es una recuca, el paisa cuelga los guallos en vez de decir que muere, y no fuma marihuana solo se traba o se tuerse.
